# Conversor-de-moedas-Python
Conversor de moedas

Um conversor de moedas simples, utilizando WxformBuilder e python.
Usa APIs para verificar a cotação das moedas usadas na aplicação.

# APIs:
- Moedas fiduciárias - https://api.frankfurter.app/latest?base=BRL&symbols=USD,EUR,GBP,CNY
- Criptomoedas - https://api.coingecko.com/api/v3/simple/price?ids=bitcoin,ethereum&vs_currencies=brl


# Tecnologias Utilizadas
- Python 3
- wxPython: Biblioteca para criação da interface gráfica nativa.
- wxFormBuilder: Ferramenta RAD para o design visual da janela.
